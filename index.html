React Jacdac - v1.0.1

React package

This package allows you to integrate Jacdac into React applications. The package exposes JDOM, a dependency-free JavaScript object model that reflects the state of the Jacdac elements and allows sending commands as well. JDOM also handles connection through WebUSB, WebBLE and other transports.

To read guides and overview documents about JDOM, go to JDOM documentation.

To browser the API documentation, use the list on this page to explore classes.

Installation

Add the react-jacdac npm package module to your project

npm install --save react-jacdac

or

yarn add react-jacdac

Hooks

useChange

This hook queries values form a JDOM element, and listens for the CHANGE event.

import { useChange } from "react-jacdac"

// query the current list of devices
const devices = useChange(bus, _ => _.devices())
